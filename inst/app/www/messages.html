iDEP 1.1: interactive & reproducible

  • Exploratory analysis, D.E.G., and pathway analyses on bulk RNA-seq data,
  • Rich, interactive visualization: clustering, KEGG pathways, genome view, etc.
  • Download HTML reports for some steps, documenting parameters and results,
  • Download dozens of publication-ready figures in minutes,
  • Download R code and data objects, which can be used in RStudio to reproduce results,
  • Install locally as an R package, which automatically downloads database,
  • Completely rewritten using Shiny modules with the Golem framework.

iDEP is still under active development and testing. We welcome any suggestions or questions. Please Email Jenny or report issues on our GitHub page.

iDEP has not been thoroughly tested.
